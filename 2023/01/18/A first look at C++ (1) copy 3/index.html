Operators

Function: Operators used to perform code

In this chapter, we will focus on the following types of operators.

Types of operators What they do
Arithmetic operators are used to handle the four operations
Assignment operators are used to assign the value of an expression to a variable
Comparison operators are used to compare expressions and return a true or false value
Logical operators are used to return a true or false value depending on the value of the expression

Comparison operators

Function: Used to compare expressions and return a true or false value

The comparison operators have the following notation.

Operator Term Example Result
== equal to 4 == 3 0
! = not equal to 4 ! = 3 1
< less than 4 < 3 0

greater than 4 > 3 1

<= less than or equal to 4 <= 3 0
= greater than or equal to 4 >= 1 1

Logical operators

Function: Used to return a true or false value depending on the value of the expression.

Logical operators have the following symbols.

Operator Term Example Result
! non !a is true if a is false, or !a is false if a is true.
&& with a && b The result is true if both a and b are true, otherwise it is false.
|| or a || b If one of a and b is true, the result is true, and if both are false, the result is false.
