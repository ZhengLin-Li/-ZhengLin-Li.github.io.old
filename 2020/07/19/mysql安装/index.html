如果出现“发生系统错误2”的提示，我的解决方法是：

在管理员命令行窗口下，进入安装的mysql的bin目录下：

1. mysql -remove

2. mysql -install

3. net start mysql

(4. net stop mysql (关闭mysql))

配置环境变量

  2. 点击系统变量的Path，编辑Path。
  3. 点击新建，把bin目录加入其中。
